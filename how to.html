How to PrintCC

To help with learning the software, heres a simple guide on how to use it!

When you open the application, you will see an popup that should look similar to below.


Heres a breakdown of how to use the application


1. A list of requirements to fulfill the equation. Click on the white box to the right of the wording to type in the needed variables

2.1. The calculate button. Press the button to calculate the total cost of your 3D print. When clicked, a .TXT file will be entered into the programs folder with the end result.

2.2. The end result being a organized list of every variable combined into the sum cost. A popup will appear after clicking, comfirming the file creation.

3. Displays the total price at the end

4.1. Input the values needed at the white box above. Incorrect answers (such as letters in questions related to money) will be ignored, but the program cannot detect 'slip-ups'

4.2 No symbols are needed for the values, just the numbers or wording requested.

5. When the .TXT is created, it will ALWAYS show up in the same folder in the program. This will likely be changed in a future version.




The program may change in future versions. Thank you again for using PrintCC!
